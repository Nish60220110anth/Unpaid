# Team Unpaid Backend

## Conventions

|Symbol|Description|
|----|----|
|**?** | Why ?|
|**!** | Note (user specific) |
|**!!**| Note (developer specific) |
|**!>**| In future|
|**!x**| Not recommended|
|**!<**| Implemented|
|**=>**| Denotes |
|**!xx**| Not Supported|

> Conventions can be mixed
>  
> Ex : **!!!>**  => Developer specific feature which need to be implemented in future
